# Jot UI 기획서

## 1. 앱 개요

**앱 이름**: Jot
**슬로건**: 흩어진 생각을 AI로 정리하고, 팀과 함께 확장하는 메모의 새로운 기준
**타겟 플랫폼**: iOS, Android

---

## 2. 디자인 시스템

## 3. 화면 구성

### 3.1 화면 목록

| 화면 ID | 화면명    | 설명                      |
| ------- | --------- | ------------------------- |
| S01     | 메모 목록 | 홈 화면, 모든 메모 리스트 |
| S02     | 메모 상세 | 메모 보기/편집 화면       |
| S03     | 메모 작성 | 새 메모 작성 화면         |
| S04     | 검색      | 메모 검색 화면            |
| S05     | 프로필    | 사용자 프로필/설정        |

### 3.2 네비게이션 구조

```
App
├── Bottom Tab Navigator
│   ├── 메모 목록 (S01)
│   ├── 검색 (S04)
│   ├── 메모 작성 (S03) - 중앙 FAB 버튼
│   ├── (예약)
│   └── 프로필 (S05)
│
└── Stack Navigator (모달)
    └── 메모 상세 (S02)
```

## 4. 화면별 상세 UI

### 4.1 S01: 메모 목록 화면

#### 레이아웃

```
┌─────────────────────────────────┐
│ [StatusBar]                     │
├─────────────────────────────────┤
│ ← Jot                    🔖  ⋯ │  ← 헤더
├─────────────────────────────────┤
│ 🔍 Search                       │  ← 검색바
├─────────────────────────────────┤
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 신규 프로젝트 기획          │ │  ← 메모 카드
│ │ 신규 프로젝트 기획을 확정   │ │
│ │ ☑ 초기 컨셉 구상           │ │
│ │ ☐ 팀원 구성 및 역할 배분   │ │
│ │ [썸네일]              12:30 │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 회의록 정리                 │ │
│ │ Q3 마케팅 캠페인 논의...    │ │
│ │                       어제  │ │
│ └─────────────────────────────┘ │
│                                 │
│              ...                │
│                                 │
├─────────────────────────────────┤
│  ≡    🔍    [+]    ▢    👤    │  ← 하단 탭바
└─────────────────────────────────┘
```

#### 구성 요소

| 요소      | 설명                    | 동작                              |
| --------- | ----------------------- | --------------------------------- |
| 헤더      | 앱 로고, 북마크, 더보기 | 북마크: 저장된 메모, 더보기: 메뉴 |
| 검색바    | 빠른 검색 입력          | 탭 시 검색 화면 이동              |
| 메모 카드 | 메모 미리보기           | 탭 시 상세 화면 이동              |
| 하단 탭바 | 네비게이션              | 각 화면으로 이동                  |

#### 메모 카드 상세

| 영역          | 내용                   | 조건        |
| ------------- | ---------------------- | ----------- |
| 제목          | 메모 제목 (최대 1줄)   | 필수        |
| 본문 미리보기 | 본문 텍스트 (최대 2줄) | 선택        |
| 체크리스트    | 체크 항목 (최대 2개)   | 있을 경우만 |
| 썸네일        | 첨부 이미지 미리보기   | 있을 경우만 |
| 타임스탬프    | 생성/수정 시간         | 필수        |

## 5. 컴포넌트 명세

### 5.1 공통 컴포넌트

| 컴포넌트       | 파일명              | 설명                |
| -------------- | ------------------- | ------------------- |
| Header         | `Header.js`         | 화면 상단 헤더      |
| BottomTabBar   | `BottomTabBar.js`   | 하단 네비게이션 탭  |
| MemoCard       | `MemoCard.js`       | 메모 목록 카드      |
| ChecklistItem  | `ChecklistItem.js`  | 체크리스트 항목     |
| SearchBar      | `SearchBar.js`      | 검색 입력 필드      |
| FloatingButton | `FloatingButton.js` | 플로팅 액션 버튼    |
| BottomToolbar  | `BottomToolbar.js`  | 편집 화면 하단 툴바 |
| ImageThumbnail | `ImageThumbnail.js` | 이미지 썸네일       |

### 5.2 MemoCard 컴포넌트 상세

```
Props:
- id: string (메모 ID)
- title: string (제목)
- content: string (본문)
- checklist: Array<{id, text, checked}> (체크리스트)
- images: Array<string> (이미지 URL)
- createdAt: Date (생성일)
- updatedAt: Date (수정일)
- onPress: function (탭 이벤트)
```

### 5.3 ChecklistItem 컴포넌트 상세

```
Props:
- id: string (항목 ID)
- text: string (항목 텍스트)
- checked: boolean (체크 상태)
- onToggle: function (체크 토글)
- onDelete: function (삭제)
- onEdit: function (편집)
```
